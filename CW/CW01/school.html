Advice: How to Maintain Quality Control over Web Design

  • Ensure consistent styles across webpages (fonts, colors).
  • Provide instructures with templates to follow to fill out.
  • Use external stylesheets to separate content from styling.
  • Test the webpage on different browsers to ensure consistency.

The Process

To maintain consistent styling, I recommend using external CSS stylesheets that all pages can reference. Having all the styles in one place ensures consistency across the entire website. This is better than allowing each teacher to write their styles because it gets rid of the possibility of using different fonts or colors which may make the website look inconsistent. Using external stylesheets also make changes easier, in comparison to making individual changes across several sections which would be time consuming.

Exploring Some HTML Tags and Styles

HTML Tags

  1. Structural Tags
    • <div>: Defines a section in an HTML document
    • <header>: Defines a header for the page
    • <footer>: Defines a footer for the page
    • <section>: Defines a section of content
  2. Text, Link, and Image Tags
    • <h1>, <h2>...<h6>: Defines headings where h1 is the biggest and h6 is the lowest
    • <p>: Defines a paragraph
    • <span>: Used to style a part of a text
    • <strong>: Used to bold text
    • <br>: Inserts a line break
    • <hr>: Inserts a horizontal line
    • <a>: Used for a hyperlink
    • <img>: Embeds an image
  3. List Tags
    • <ul>: Unordered list
    • <ol>: Ordered list
    • <li>: A list item
    • <dl>: Description list
    • <dt>: A term in a dl
    • <dd>: The description of the dt
  4. Table Tags
    • <table>: Defines a table
    • <tr>: A table row
    • <td>: A table cell
    • <th>: A table header cell

CSS Tags

  1. Text Styles
    • color: To change text color
    • font-family: Defines the font
    • font-size: To change text size
    • font-weight: To set the thickness of the font (bold, normal)
    • text-align: To align the text (left, center, right)
  2. Box Styles
    • width: Defines the width
    • height: Defines the height
    • padding: Defines the space between the content and the border
    • margin: Defines the space outside the border
    • background-color: Defines the background color of the box
